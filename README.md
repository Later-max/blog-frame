#### 使用的工具：vue+Element-ui+node+mongodb

## 1. 环境配置 

 - 注：本地需要先下载mongodb数据库，不然node无法跑起
 
 - 脚手架3及其以上**创建项目**
  **创建`vue create element-admin`**
 - 添加**element**框架
 **`vue add element`** 
 - **node部署**---对应文件server-index.js
 **`npm i express@next mongoose cors`**
 - 安装**nodemon**
 **`npm i -g nodemon`**
 - **前端启动**本地服务
 **`npm run serve`**
 - **后端启动**本地服务
 **`nodemon server`**
